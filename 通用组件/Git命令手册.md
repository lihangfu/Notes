# 常用操作

## 提交
        git commit  #提交到当前分支

## 新建分支
        git branch <name>   #新建分支

        git checkout <name> #选择分支

        git checkout -b <name>  #新建并选择分支

## 合并分支merge
        git merge <name>  #其他分支合并到当前分支

## 合并分支rebase
        git rebase <name>   #当前分支并入其它分支，当前分支的上条纪录只是其它分支的一条纪录
